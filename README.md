# SmartCart 🛒

SmartCart is a Flask web app for discovering weekly grocery deals and saving favorite stores.

## Features (Planned)

- User accounts and login
- Save favorite grocery stores
- View the latest weekly ads
- Personalized dashboard
- Simple and clean UI

## Tech Stack

- Python
- Flask
- SQLite
- Flask-Login
- HTML / CSS / JavaScript
